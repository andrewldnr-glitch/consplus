Нужен фон в стиле референса: **dark + premium + purple gradient + subtle grid + мягкие волны/ауры**, без текста и логотипов.

**Background 4K:**

> Premium minimalistic dark background for a mobile learning app, deep charcoal/black base, subtle geometric grid barely visible, soft purple-to-violet gradient aurora, gentle glow, abstract wave shapes, smooth blur, modern fintech aesthetic, no text, no logos, high resolution, clean, elegant, 4K, mobile wallpaper, safe margins for UI.
