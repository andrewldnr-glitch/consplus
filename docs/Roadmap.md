## MVP (2–4 спринта)

### Sprint 1 — Foundation + Premium UI

- [ ] каркас SPA, hash-router
- [ ] дизайн‑токены + фон (dark + purple)
- [ ] bottom nav + topbar + safe area
- [ ] Splash + Onboarding
- [ ] Home (hero + KPI)

**DoD:** приложение красиво открывается в Telegram, навигация работает.

### Sprint 2 — Knowledge Base

- [ ] структура данных `CONTENT`
- [ ] экран KB (поиск, focus toggle)
- [ ] экран Package detail (табы уровней, “что входит”)
- [ ] deeplinks и share
- [ ] словарь терминов

**DoD:** пользователь может ответить “что входит” без скролла кверху и без таблиц.

### Sprint 3 — Tests Engine

- [ ] Tests Hub + preview
- [ ] Test runner (single/multi/boolean/fill/matching/ordering)
- [ ] Results + разбор + ссылки на повторение
- [ ] генераторы “packageByBank” и “upgradeDiff”
- [ ] пример банка вопросов (30 вопросов)

**DoD:** нет логических ошибок в тестах (single только при единственном ответе).

### Sprint 4 — Stats + Premium + Polish

- [ ] Stats: история, график 7 дней, streak
- [ ] Profile: настройки, reset, docs links
- [ ] Paywall screen (демо)
- [ ] QA регрессия + улучшение микрокопи + пустые состояния

**DoD:** готово к деплою на GitHub Pages, UX приятный, стабильный.
